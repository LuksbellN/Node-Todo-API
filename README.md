# Node-Todo-API
Um projeto de práticas de Rest api's com uma TODO list registrada em um banco de dados local PostgreSQL
Projeto realizado com o auxílio da B7WEB cursos com o modulo de NodeJS

# Install
npm install

# Run project
npm run start-dev
